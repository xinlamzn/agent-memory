== PydanticAI Integration

PydanticAI provides type-safe agent development with Pydantic models.

=== Installation

[source,bash]
----
pip install neo4j-agent-memory[pydantic-ai]
----

=== Basic Usage

[source,python]
----
from dataclasses import dataclass
from pydantic_ai import Agent
from neo4j_agent_memory import MemoryClient, MemorySettings

@dataclass
class AgentDeps:
    memory: MemoryClient
    session_id: str

# Create the agent
agent = Agent(
    "openai:gpt-4o-mini",
    deps_type=AgentDeps,
    system_prompt="You are a helpful assistant with memory capabilities."
)

@agent.tool
async def remember_preference(ctx, category: str, preference: str) -> str:
    """Store a user preference."""
    await ctx.deps.memory.long_term.add_preference(
        category=category,
        preference=preference,
        importance=0.8
    )
    return f"Remembered: {preference}"

@agent.tool
async def recall_context(ctx, query: str) -> str:
    """Recall relevant context from memory."""
    context = await ctx.deps.memory.get_context(
        query=query,
        session_id=ctx.deps.session_id
    )
    return context

async def main():
    settings = MemorySettings()

    async with MemoryClient(settings) as memory:
        deps = AgentDeps(memory=memory, session_id="user-123")

        # Run the agent
        result = await agent.run(
            "Remember that I prefer vegetarian food",
            deps=deps
        )
        print(result.data)
----

=== With Streaming

[source,python]
----
from pydantic_ai import Agent
from sse_starlette.sse import EventSourceResponse
from fastapi import FastAPI

app = FastAPI()

@app.post("/chat")
async def chat(message: str, session_id: str):
    async def event_generator():
        async with MemoryClient(settings) as memory:
            # Store user message
            await memory.short_term.add_message(
                session_id=session_id,
                role="user",
                content=message
            )

            # Get context
            context = await memory.get_context(
                query=message,
                session_id=session_id
            )

            # Stream response
            deps = AgentDeps(memory=memory, session_id=session_id)
            async with agent.run_stream(message, deps=deps) as stream:
                full_response = ""
                async for chunk in stream.stream_text():
                    full_response += chunk
                    yield {"event": "message", "data": chunk}

            # Store assistant response
            await memory.short_term.add_message(
                session_id=session_id,
                role="assistant",
                content=full_response
            )

    return EventSourceResponse(event_generator())
----

=== Automatic Trace Recording

The `record_agent_trace()` function automatically captures a PydanticAI agent's reasoning trace from a `RunResult`, including all tool calls and their results.

[source,python]
----
from pydantic_ai import Agent
from neo4j_agent_memory import MemoryClient
from neo4j_agent_memory.integrations.pydantic_ai import record_agent_trace

agent = Agent(
    "openai:gpt-4o-mini",
    system_prompt="You are a helpful assistant."
)

@agent.tool
async def search_web(query: str) -> str:
    """Search the web for information."""
    return f"Results for: {query}"

async def main():
    async with MemoryClient(settings) as memory:
        # Run the agent
        result = await agent.run("Search for Python tutorials")

        # Automatically record the trace
        trace = await record_agent_trace(
            memory.procedural,
            session_id="user-123",
            result=result,
            task="Search for Python tutorials",
            include_tool_calls=True,
            generate_embeddings=False,
        )

        print(f"Recorded trace: {trace.id}")
        print(f"Steps: {len(trace.steps)}")
        print(f"Success: {trace.success}")
----

The function extracts:

* **Reasoning steps** - Each model request/response pair becomes a step
* **Tool calls** - Arguments, results, and timing are captured
* **Success/failure** - Detected from tool results (errors, exceptions)
* **Timing** - Duration calculated from result timestamps

Parameters:

[cols="1,2,1"]
|===
|Parameter |Description |Default

|`procedural_memory`
|The ProceduralMemory instance
|Required

|`session_id`
|Session to associate the trace with
|Required

|`result`
|PydanticAI RunResult to record
|Required

|`task`
|Description of the task (extracted from first user message if not provided)
|`None`

|`include_tool_calls`
|Whether to record tool calls
|`True`

|`generate_embeddings`
|Generate embeddings for steps
|`False`
|===

== LangChain Integration

LangChain is a framework for building LLM-powered applications.

=== Installation

[source,bash]
----
pip install neo4j-agent-memory[langchain]
----

=== As Chat Memory

[source,python]
----
from langchain_core.messages import HumanMessage, AIMessage
from neo4j_agent_memory.integrations.langchain import Neo4jChatMessageHistory

# Create chat history backed by Neo4j
history = Neo4jChatMessageHistory(
    session_id="user-123",
    memory_client=memory_client
)

# Add messages
await history.aadd_messages([
    HumanMessage(content="Hello!"),
    AIMessage(content="Hi there! How can I help you?")
])

# Get messages
messages = await history.aget_messages()
----

=== With ConversationChain

[source,python]
----
from langchain.chains import ConversationChain
from langchain.memory import ConversationBufferMemory
from langchain_openai import ChatOpenAI

# Create memory backed by Neo4j
neo4j_history = Neo4jChatMessageHistory(
    session_id="user-123",
    memory_client=memory_client
)

memory = ConversationBufferMemory(
    chat_memory=neo4j_history,
    return_messages=True
)

# Create chain
chain = ConversationChain(
    llm=ChatOpenAI(),
    memory=memory
)

# Use the chain
response = await chain.ainvoke({"input": "What's my name?"})
----

=== As Retriever

[source,python]

== LlamaIndex Integration

LlamaIndex is a data framework for connecting LLMs with external data.

=== Installation

[source,bash]
----
pip install neo4j-agent-memory[llamaindex]
----

=== As Chat Store

[source,python]
----
from neo4j_agent_memory.integrations.llamaindex import Neo4jChatStore

# Create chat store
chat_store = Neo4jChatStore(
    memory_client=memory_client
)

# Use with chat engine
from llama_index.core.memory import ChatMemoryBuffer

memory = ChatMemoryBuffer.from_defaults(
    chat_store=chat_store,
    chat_store_key="user-123"
)
----

=== As Vector Store

[source,python]
----
from neo4j_agent_memory.integrations.llamaindex import Neo4jMemoryVectorStore

# Create vector store backed by Neo4j memory
vector_store = Neo4jMemoryVectorStore(

from llama_index.core import VectorStoreIndex

index = VectorStoreIndex.from_vector_store(vector_store)

query_engine = index.as_query_engine()

== CrewAI Integration

CrewAI enables multi-agent orchestration with specialized roles.

=== Installation

[source,bash]
----
pip install neo4j-agent-memory[crewai]
----

=== Shared Memory for Crew

[source,python]
----
from crewai import Agent, Task, Crew
from neo4j_agent_memory.integrations.crewai import Neo4jCrewMemory

# Create shared memory
crew_memory = Neo4jCrewMemory(
    memory_client=memory_client,
    crew_id="research-crew"
)

# Create agents with shared memory
researcher = Agent(
    role="Researcher",
    goal="Find relevant information",
    memory=crew_memory
)

writer = Agent(
    role="Writer",
    goal="Write compelling content",
    memory=crew_memory
)

# Create crew
crew = Crew(
    agents=[researcher, writer],
    tasks=[...],
    memory=crew_memory  # Shared memory
)

# Run crew
result = await crew.kickoff()
----

[source,python]

from neo4j_agent_memory.integrations.crewai import Neo4jAgentMemory

researcher_memory = Neo4jAgentMemory(

researcher = Agent(

== Custom Integration

You can integrate neo4j-agent-memory with any framework using the core API.

=== Memory Context Provider

[source,python]
----
class MemoryContextProvider:
    def __init__(self, memory_client: MemoryClient, session_id: str):
        self.memory = memory_client
        self.session_id = session_id

    async def get_context(self, query: str) -> str:
        """Get relevant context for a query."""
        return await self.memory.get_context(
            query=query,
            session_id=self.session_id,
            include_short_term=True,
            include_long_term=True,
            include_procedural=True
        )

    async def store_interaction(self, user_msg: str, assistant_msg: str):
        """Store a conversation turn."""
        await self.memory.short_term.add_message(
            session_id=self.session_id,
            role="user",
            content=user_msg
        )
        await self.memory.short_term.add_message(
            session_id=self.session_id,
            role="assistant",
            content=assistant_msg
        )
----

=== Tool Wrapper

[source,python]
----
def create_memory_tools(memory: MemoryClient, session_id: str):
    """Create memory tools for any agent framework."""

    async def search_memory(query: str) -> str:
        """Search across all memory types."""
        return await memory.get_context(
            query=query,
            session_id=session_id
        )

    async def remember_fact(subject: str, predicate: str, obj: str) -> str:
        """Store a fact in long-term memory."""
        await memory.long_term.add_fact(
            subject=subject,
            predicate=predicate,
            object=obj
        )
        return f"Remembered: {subject} {predicate} {obj}"

    async def remember_preference(category: str, preference: str) -> str:
        """Store a user preference."""
        await memory.long_term.add_preference(
            category=category,
            preference=preference
        )
        return f"Remembered preference: {preference}"

    return {
        "search_memory": search_memory,
        "remember_fact": remember_fact,
        "remember_preference": remember_preference,
    }
----

== Best Practices

=== Session Management

[source,python]
----
# Use consistent session IDs across frameworks
session_id = f"user-{user_id}-{conversation_id}"

# Share the same memory client
async with MemoryClient(settings) as memory:
    # Use with multiple integrations
    langchain_history = Neo4jChatMessageHistory(session_id, memory)
    pydantic_deps = AgentDeps(memory=memory, session_id=session_id)
----

=== Error Handling

[source,python]
----
from neo4j_agent_memory import ConnectionError, ExtractionError

try:
    context = await memory.get_context(query)
except ConnectionError:
    # Fall back to no context
    context = ""
except ExtractionError:
    # Log and continue without extraction
    logger.warning("Entity extraction failed")
----

=== Performance

* Reuse `MemoryClient` instances across requests
* Use connection pooling for high-throughput applications
* Consider disabling extraction for bulk operations
* Use batch extraction for processing multiple documents
* Enable streaming extraction for large documents

=== Observability

Enable tracing for debugging and monitoring agent memory operations:

[source,python]
----
from neo4j_agent_memory import MemorySettings, ObservabilityConfig, TracingProvider

# Enable Opik tracing for LLM observability
settings = MemorySettings(
    observability=ObservabilityConfig(
        enabled=True,
        provider=TracingProvider.OPIK,
        project_name="my-agent",
        track_llm_calls=True,
        track_extraction=True,
        track_memory_ops=True,
    )
)

# All memory operations will now be traced
async with MemoryClient(settings) as memory:
    # Operations are automatically traced
    context = await memory.get_context(query="user query", session_id="123")
----

Traces include:
* Memory read/write operations with timing
* Entity extraction pipeline stages
* LLM API calls with token counts
* Error tracking and debugging info

=== Entity Deduplication

Enable deduplication to prevent duplicate entities during ingest:

[source,python]
----
from neo4j_agent_memory import MemorySettings, DeduplicationConfig

settings = MemorySettings(
    deduplication=DeduplicationConfig(
        enabled=True,
        embedding_threshold=0.92,
        create_same_as=True,  # Link similar entities
    )
)

# Entities are automatically deduplicated during extraction
async with MemoryClient(settings) as memory:
    await memory.long_term.add_message(
        session_id="123",
        role="user",
        content="I met John Smith at Acme Corp"  # Deduped with existing "John Smith"
    )
----
